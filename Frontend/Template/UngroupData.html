FORMULAS & DEFINITIONS

  • Mean: X̄ = Σx / n
  • Sample Variance: s² = Σ(xi - X̄)² / (n-1)
  • Population Variance: σ² = Σ(xi - X̄)² / n
  • Sample Standard Deviation: s = √s²
  • Population Standard Deviation: σ = √σ²
